# Retro Bakes — Premium Bakery Landing Page

A modern premium bakery landing page designed and developed for Retro Bakes, a dessert and cake brand based in Vijayanagar, Bengaluru.

This project was built to create a luxury online presence for a local bakery brand with a strong focus on:
- modern UI/UX
- mobile-first responsiveness
- premium visual branding
- customer engagement
- smooth user experience

---

## ✨ Features

- Premium modern landing page
- Fully responsive design
- Elegant hero section
- Featured desserts showcase
- Customer reviews section
- Instagram-style gallery
- Custom cakes section
- Store visit/contact section
- Smooth animations & transitions
- WhatsApp CTA integration
- SEO-friendly structure

---
