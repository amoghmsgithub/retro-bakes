# Retro Bakes Frontend

Premium bakery landing page frontend built for Retro Bakes, a dessert and custom cake brand based in Vijayanagar, Bengaluru.

This frontend focuses on delivering:
- luxury UI/UX
- smooth animations
- responsive experience
- premium visual branding
- modern customer-focused design

---

# ✨ Features

- Premium hero section
- Responsive navigation bar
- Featured desserts showcase
- Customer reviews section
- Instagram-style gallery
- Custom cake showcase
- Smooth animations
- Mobile-first responsive layout
- Elegant typography
- Modern bakery aesthetic

---

# 🛠️ Frontend Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- CRACO
- PostCSS
